# PNW Stump Pros

## PNW Stump Pros - stump grinding service landing page - [https://pnwstumpprops.vercel.app](https://pnwstumpprops.vercel.app/?utm_source=github&utm_medium=readme)

### Installation & control

```sh
npm i
```

### Dev-mode

```sh
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


## Tech Stack

- React 19
- Next.js 16 (App Router) + SSR
- Tailwind CSS v4
- Heroicons
- TypeScript
